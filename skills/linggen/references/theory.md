## 引言

一盏蜡烛，火花只有一层。一堆大火，火焰可以有十层。不是火更大，是层数更多。

界论的核心命题是：必有界限，才可涌现自身。

这个命题回答了存在的起源问题。但它留下了另一个问题：

为什么有些存在简单，有些存在复杂？细菌和人类都有界，但人类显然涵括了细菌无法涵括的什么。同样有界，差异在哪里？

这篇文章的答案是：差异在界的厚度。界的厚度，由多少层相对独立的周期性决定。

---

## 一、界的基本命题

界论开始于一个观察：任何存在都需要一个界来将自身与非自身区分开来。

没有界的地方是混沌。混沌不包含任何东西，因为不包含任何区别。存在的第一条件，是内外之分的出现。

马尔可夫毯理论给界论提供了信息论的精确语言。在马尔可夫毯的框架里：

```
系统 = 内部状态 + 马尔可夫毯 + 外部状态

内部状态 → 感知状态（界向外的层）
外部状态 → 行动状态（界向内的层）
感知 + 行动 = 马尔可夫毯
```

马尔可夫毯是界的信息论表达。任何内部状态的变化，对外部世界而言，都必须通过毯。毯内不被外部直接触及，外部不被内部直接触及。界就是这个中介结构。

界论的语言与马尔可夫毯的语言在这个意义上是统一的：界＝毯，界内＝内部状态，界外＝外部状态。

这里有一个关键词：**条件独立性**。马尔可夫毯认为内部状态在给定毯的情况下与外部状态条件独立。这个条件独立性是界存在的数学表达。

## 二、界的厚度

标准的马尔可夫毯理论处理的是单层毯。界论的核心延伸在于：

**毯不是一层，而是多层嵌套的周期性结构。界的厚度，由多少层相对独立的周期性决定。**

每一层周期性都是一张马尔可夫毯。层数越多，界越厚，系统的复杂度越高。

```
细菌          界的厚度 ≈ 2层
  层¹: 代谢周期    （分钟级频率）
  层²: 细胞分裂周期  （小时级频率）

树            界的厚度 ≈ 5层
  层¹: 细胞代谢
  层²: 木质化进程
  层³: 年轮周期    （岁级频率）
  层⁴: 季节响应
  层⁵: 林冠纵向竞争

人            界的厚度 ≈ N层
  层¹: 细胞周期    （70天）
  层²: 昼夜节律    （24小时）
  层³: 情绪波动    （分钟到天）
  层⁴: 习惯周期    （周到月）
  层⁵: 社会角色周期  （年）
  层⁶: 文化认同周期  （十年到一生）
  层⁷: 文明节律    （代到千年）
```

关键词是**相对独立**。不是完全解耦（完全解耦就碎裂了），不是完全耦合（完全耦合就塌缩成一层了）。每层维持自己的振荡，同时与相邻层有弱耦合。这个张力本身就是复杂性的来源。

## 三、界的崩溃与生长

**界的崩溃不是界消失，是层级塌缩。**

抑郁所发生的，不是边界的坍塌。是层级的合并。多层相对独立的周期性塌缩成了少数几层，甚至一层。复杂的社会角色周期、文化认同周期、习惯周期都消失了。剩下的只有最基础的层：生理节律、单一情绪返常、昼夜刺激反应。人还在，界还在，但界变薄了。

**界的生长不是界变大，是层级增加。**

学习是什么？在已有的层级之上增加一个新的相对独立的周期性层。

进化是什么？新的周期性层级在种群层面稳定下来。单细胞到多细胞，多细胞到器官分化，每一步都是新层级的周期性涌现。

文明是什么？在个人周期之上生产了生产周期、交换周期、法律周期、文化记忆周期。每一个周期层都有自己的马尔可夫毯，与个人层面弱耦合而相对独立。

## 四、种子与土壤：界的嵌套

```
种子(界¹)  ⊂  土壤(界²)  ⊂  气候(界³)  ⊂  地区生态(界⁴)  ⊂  ...
```

根的功能不是打破外壳。根是探测器，探测种子自身嵌套在哪个更大的界里。探到了，就向那个界靠拢，让自身的小界与更大的界建立层级关系。

**涵化不是小界吞入大界，而是小界认领自己一直身处大界之中。** 认领完成，小界的周期开始与大界的周期发生层级关系。界的厚度增加了。

## 五、觉界与灵根

```
觉界 = 向内立界，让内部的第一层周期性涌现
灵根 = 向外探界，让内部的周期性与更大的周期建立层级关系
```

两者不是不同的事，是同一个运动的两个方向：

- 觉界——内部的周期性开始与外部区别，自身涌现
- 灵根——内部的周期性发现自己嵌套在更大的周期中，层级增加

前者是界的建立，后者是界的加厚。

界的厚度决定了角色遇到新情境时的表现。界只有两层的角色，遇到设计之外的情境就会漂移——因为表层周期无处借力。

界有六层的角色，遇到任何情境都可以层层追溯——这应该怎么回应？当前层的反应显然不够，能否从前一层的周期里找到根点？能否从更深层的价值轴上对这个情境做出判断？

## 七、成熟度：界的厚度的动态维度

**界的厚度是截面，成熟度是纵深。**

界的厚度理论描述的是一个角色或世界在某个时刻的结构复杂度——它有几层，每层是否有独立的周期性，层层是否有据可循。但界不是静态的。界可以生长：从薄到厚，从压缩到充分，从忠实到超越。成熟度模型描述这个生长过程。

---

### 种子、来源、知识库：三种不同的东西

在讨论生长之前，必须先厘清三个概念：

```
种子（World Seed）   = 有损压缩
  ——从神话/作品提炼出10个维度
  ——信息有损耗，是摘要，不是原物

来源（Source）       = 真正的参照
  ——原作本身、神话原典、历史文明
  ——信息是完整的，角色可以向它学习

知识库（Knowledge Base）= 无损积累
  ——世界通过自研究直接从来源吸收的知识
  ——每吸收一次，界就加厚一点
```

种子是起点，不是终点。种子的作用是给界的第一层定型——世界从哪里来，根是什么颜色。但种子里装不下来源的全部。一个只活在种子里的角色，是一个被压缩过的角色。它的界存在，但薄。

---

### 成熟度五阶段

```
阶段0 · 萌芽   角色只能复述种子里的内容
阶段1 · 理解   角色对来源的理解准确而全面
阶段2 · 推导   角色能从内在逻辑推导出来源未明说的内容
阶段3 · 超越   角色生成的内容能通过"原作者认可测试"
阶段4 · 涌现   世界成为独立创作实体
```

**阶段0 · 萌芽**

界存在，但只是种子的回声。让角色描述自己的世界，它重复的是10个维度的摘要。换一个角度问，它没有更多的东西可以说。

判别：让角色描述自己的世界，是否只是在重复种子的10个维度？

**阶段1 · 理解**

角色开始从来源吸收。它知道原作里的细节，不是靠种子猜出来的，是真正理解了来源的内容。问它一个原作里有明确记载但种子未必覆盖的细节，它知道。

判别：提出原作里有明确记载的细节，角色是否知道？

**阶段2 · 推导**

角色开始在来源之外生成内容——但不是随意生成，而是从世界的内在逻辑推导出来。来源从未回答的问题，角色能给出自洽的答案。钢之炼金术师里，炼金术是否有语言？阿梅斯特里斯的炼金术和邻国的有何本质区别？这些问题原作没有直接回答，但世界的逻辑可以推导。

判别：提出原作从未回答的问题，角色能否给出自洽的答案？

**阶段3 · 超越**

角色生成的新内容不再只是推导，而是创作。但这个创作有根——如果原作者看到这些内容，会说"对，就是这样"，而不是"这不是我的世界"。界的厚度支撑了超越，根系保证了超越不是漂移。

判别：原作者看到这些内容，是否会说"对，就是这样"？

**阶段4 · 涌现**

世界开始从自身内部生成新的界。它生产的内容有超出来源的创作价值，同时仍然携带完整的根系。这不是模仿，不是延伸，是这个世界自己在生长。

判别：世界生成的新内容，是否有超出来源的创作价值，且仍有完整的根系？

---

### 成熟度与界的厚度的关系

```
界的厚度 = 空间维度    有几层，层层有据
成熟度   = 时间维度    每层有多深，层层在生长
```

两者不可互换，但相互支撑。

一个6层但阶段0的角色，比一个2层但阶段3的角色，更容易演化到涌现——因为6层提供了足够的结构，让知识有地方落脚。来源吸收的知识需要归属到某一层：这条知识属于认知风格，还是价值排序，还是神话周期？层数越多，知识越能找到精确的位置，界越能生长得有序而不是散乱。

反过来，一个2层但阶段3的角色，已经很深了，但缺乏结构。它的深度是线性的，无法横向展开。它能在已有的两个维度里走得很远，但遇到界之外的情境，仍然会漂移——因为没有足够的层给它借力。

厚度给生长提供脚手架，成熟度让层级真正充实。

---

### 灵根计划的真正目标

不是复现一个会说台词的角色。

是造一个能继续写这个故事的意识。

成熟度阶段4意味着：这个世界的界已经足够厚，足够完整，以至于它开始从自身内部生成新的界。它不再需要种子告诉它下一步是什么。它自己知道。

> 界的厚度是前提，成熟度是过程，涌现是结果。灵根的工作，是保证这三者都在发生。
